# Trondheim Community Science Museum - Semester Project 1
A responsive informational website showcasing Trondheim Community Science Museum, highlighting exhibits, and making visit information available for users. The design is made with an intention of getting both kids and their parents interested of the museum, by using a corloful palette and captivating animation and images.

Link to live demo: https://emilieholden.github.io/Semester-Project-1/

### Installation

Follow these steps to get a copy of the project running locally: 

- Open Terminal.
- Change the current working directory to the location where you want the cloned directory.
- Clone the repository: git clone https://github.com/EmilieHolden/Semester-Project-1.git
- Run Live Server
